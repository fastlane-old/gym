fastlane documentation
================
# Installation
```
sudo gem install fastlane
```
# Available Actions
### test
```
fastlane test
```
Run bundle install and tests
### release
```
fastlane release
```
Does everything that's needed for a release, this includes running tests and verifying the GitHub release
